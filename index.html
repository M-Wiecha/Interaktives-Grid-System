Das CSS Grid-System Labor

CSS Grid ist ein mächtiges, zweidimensionales Layout-System. Es steuert Spalten und Zeilen gleichzeitig!

Wichtige Grid-Befehle zum Testen:

1. Spalten definieren

Nutze grid-template-columns, um die Anzahl und Breite der Spalten festzulegen:

Beispiel: repeat(3, 1fr) (3 gleiche Spalten) oder 200px 1fr.

2. Abstände (Gaps)

Mit gap definierst du die Abstände zwischen den Zeilen und Spalten völlig ohne Margin:

Werte: z.B. 10px, 20px oder zwei Werte für [Zeile] [Spalte].

3. Elemente strecken (Spanning)

Füge der Klasse .item-featured im CSS die Eigenschaft grid-column: span 2; hinzu. Die Box nimmt zwei Spalten ein!
